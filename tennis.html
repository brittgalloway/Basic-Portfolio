Tennis News

  • Github :
    https://github.com/brittgalloway/tennisNews
  • Live App :
    https://tennis-news11-24.herokuapp.com/
  • This tennis new app is a full stack web page that scrapes articles from the offical ATP Tour website. You can view the Headline, Thumbnail Image, Category, and Orignal Post Date. Along side each article, you can post comments, and read previous comments. If you want, you can click the headline to read the actual article.
  • Tools: Express, Express-handlebars, Mongoose, Cheerio, Axios, SASS, Bulma UI Framework
  • FUTURE DEVELOPEMENTS:

  • Can delete comments
  • Articles are scraped as new ones are added to the source website
